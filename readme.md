# piig.me

This is a custom url shortener project.

## Running Project

1. Install:
    - [Git](https://git-scm.com/)
    - `$ npm i nodemon -g`
2. In terminal:
    - `$ git clone https://github.com/CITi-UFPE/piigme && cd piigme && npm i && npm run start`
3. Access:
    - http://localhost:3333

Done!
